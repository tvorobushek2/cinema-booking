\# 🎬 Cinema Booking System



Система бронирования мест в кинотеатре 



\## 👥 Команда



\- \*\*Илья\*\* — Frontend, API чтения, кэширование (Redis)

\- \*\*Кирилл\*\* — Backend, API бронирования, транзакции, фоновые задачи (Celery)



\## 📋 Описание



Веб-приложение для бронирования мест в кинотеатре с:

\- Визуализацией зала (схема мест)

\- Блокировкой мест на время бронирования (15 минут)

\- Защитой от race condition (транзакции + FOR UPDATE)

\- Автоматическим освобождением неоплаченных броней

\- Кэшированием списка сеансов
